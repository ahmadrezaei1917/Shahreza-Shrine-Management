# Shahreza Holy Shrine Servant Management System 🕌✨

A modern, administrative web application designed for registering, organizing, and managing servant (Khademin) shifts at the **Shahreza Holy Shrine** (امامزاده شاهرضا). 

This project was successfully developed as our **Associate Degree (Kardani) Final Project**.

> ⚠️ **Note:** This project is a fully functional prototype and academic demo. It contains mock data for testing purposes and was not deployed to production.

---

## 🎨 UI/UX & Design Style
The front-end features a custom-designed **Glassmorphism (Liquid Glass)** style inspired by modern iOS UI aesthetics, fully built with **Bootstrap 5** for a responsive and clean layout.
